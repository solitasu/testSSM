设置highLineWeekDay属性 ture 或 false 可指定是否高亮周末 。 如：onclick="WdatePicker({highLineWeekDay:true})"

设置readOnly属性 true 或 false 可指定日期框是否只读

是否显示清空按钮和今天按钮,可以根据需要进行自定义,它们分别对应 isShowClear 和 isShowToday 默认值都是true

设置日期起始值、 固定值

1.当日期框为空值时固定起始值 如： onclick="WdatePicker({startDate:'1980-05-01'})"

2.当日期框无论是何值,始终使用 1980-05-01 做为起始日期 如： onclick="WdatePicker({startDate:'1980-05-01',alwaysUseStartDate:true})"

设置动态参数 如：onclick="WdatePicker({startDate:'%y-%M-01 00:00:00',dateFmt:'yyyy-MM-dd HH:mm:ss',alwaysUseStartDate:true})"/

yMdHmswW分别代表年月日时分秒星期周,你可以任意组合这些元素来自定义你个性化的日期格式.

自动纠错0 提示(默认) 1 自动纠错 2 标记 -1 禁用自动纠错 如：onclick="WdatePicker({errDealMode:1})"

最大可选择时间 如：maxDate:'%y-%M-%d' 或 maxDate:'2017-05-01'

多语言 繁体中文：onclick="WdatePicker({lang:'zh-tw'})" 英文：lang:'en' 简体中文：lang:'zh-cn' 根据浏览器的语言自动选择语言：lang='auto'

设置日期最大、最小值 如： onclick="WdatePicker({skin:'whyGreen',dateFmt:'yyyy-MM-dd HH:mm:ss',minDate:'2008-03-08 11:30:00',maxDate:'2008-03-10 20:59:30'})" value="2008-03-09 11:00:00"
